Anime Recommender

Instructions:

To get recommendations:

  1. Enter the name of an Anime in the "Search by Name" form. This will return a table of all anime in the database with similar names. Click the "Recommend" button for recommendations.
  2. You can also generate a Random Anime
  3. Or list the top 100 Anime TV or Movies in the database by Ratings.
  4. For each Anime there will be an associated "Recommender" button. Use this to generate similar Anime using a content based filtering. For more information on the anime, follow the "MAL Link" to the myanime.net webpage for that anime.
