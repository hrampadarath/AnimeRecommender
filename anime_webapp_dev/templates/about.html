About

This web app makes recommendations based upon data obtained from MyAnimeList that was shared on Kaggle. Please note the data is from 2016 and would be a bit out of date. The database would be updated soon.

This recommender was made using Python, Sklearn, Pandas and Flask. If you are interested on how the recommendations are made, please have a look at my github repo with the original Python codes.
